\textbf{几何感知3D场景特征建模方法（GeoGatedAttention）} 
\textit{（核心开发者，发明专利受理）} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{问题：} 在复杂几何结构（曲率突变、边界不连续）下，传统模型难以准确建模局部结构特征。
    \item \textbf{方法：} 设计几何门控注意力机制（GeoGatedAttention），融合曲率、法向量与边结构信息，对特征权重进行动态调制。
    \item \textbf{结果：} 在复杂3D结构建模任务中误差降至\textbf{2.69\%}（$\rho=0.97$），推理延迟\textbf{20ms}，具备实时系统部署潜力。
    \item \textbf{工程：} 实现从几何特征提取、图结构构建到模型训练的完整流程，支持非结构化网格与复杂场景输入。
    \item \textbf{应用场景：} 适用于点云特征建模、结构理解与3D场景建模等感知类任务。
\end{itemize}
