\textbf{UNOP：面向长时序时空预测的无监督神经算子} 
\textit{(一作，CCF-A 审稿中)} 
\hfill 2025.9 -- 至今
\begin{itemize}
    \item \textbf{问题定义：} 长时序3D预测在复杂几何下易发散（>20 step），传统PINN在3D场景存在显存爆炸与梯度不稳定问题。
    \item \textbf{方法设计：} 提出基于积分守恒形式的一致性约束（LIPE），以替代传统基于微分残差的PINN训练范式，降低对高质量CFD标签依赖，并显著减少训练显存开销。
    \item \textbf{结果：} 实现多工况长时间稳定外推（50-step stable rollout，无发散），误差累积增长受控，3D外流场误差4.06%，物理守恒误差降低至 $2.9\times10^{-3}$。
    \item \textbf{工程实现：} 构建完整训练与数据Pipeline，支持非结构化3D网格输入与多场耦合训练。
    \item \textbf{泛化能力：} 方法可迁移至复杂时空系统建模任务，包括多尺度PDE演化建模与高维动力系统预测。从数值方法角度，本工作构建了一类基于积分一致性的神经算子离散格式。
\end{itemize}

% ================================================================
\textbf{几何感知AI模型的汽车气动性能快速预测} 
\textit{(核心开发者，发明专利已受理)} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{任务背景：} 面向复杂3D气动外形建模中局部几何突变导致预测失准的问题。
    \item \textbf{工业级Benchmark与痛点定位：} 在真实汽车工业数据集（DrivAerNet++）和 ShapeNet-Car 上复现并测试了多款前沿神经算子（Transolver等）。发现现有模型在处理溜背/三厢、光滑/细节底盘等\textbf{几何突变区域}时，难以捕捉局部边界层效应，导致预测误差飙升。
    \item \textbf{核心算法创新 (GeoGatedAttention)：} 为解决上述痛点，设计了\textbf{几何门控注意力机制}。通过底层脚本提取网格的边缘曲率($tanh$缩放)、法向量夹角($cos\theta$)与边长，作为图结构中的 \texttt{edge\_geom\_feat} 动态调制 Attention 权重。让模型自适应感知“折角”与“平滑面”。
    \item \textbf{工程指标与业务价值：} 气动阻力系数 $C_d$ 预测误差降至 \textbf{2.69\%}，相关系数 $\rho = 0.97$。单次推理 \textbf{20ms}，替代传统CFD 6-12小时的计算，将“设计-仿真”迭代周期从\textbf{天级缩短至分钟级}。
\end{itemize}

\begin{itemize}
    \item \textbf{挑战：} 工业现场数据极度混乱，水轮机几何（.glb/.stl）存在非流形破损；流场速度（CSV点云）与叶片压力场坐标系错配，完全无法直接送入深度学习框架。
    \item \textbf{3D数据问题解决与Pipeline构建：} 
        1) 拓扑修复：利用 PyVista 提取特征边定位孔洞，结合 Meshlab/Blender 完成流形网格封闭；
        2) 网格划分：调用 TetGen 将表面转换为四面体体网格；
        3) 多源融合：编写基于 KD-Tree 的 IDW(反距离权重) 算法，实现流场/结构场多源数据在统一网格空间的对齐与映射。
    \item \textbf{图数据重构与跨域泛化：} 编写自定义 PyG Dataset，批量提取 SDF、法向量，构建 \texttt{radius\_graph} 图拓扑。在\textbf{不改变网络架构}的前提下，将汽车外流场模型迁移至水轮机内流场，成功完成 FSI（流固耦合）的第一步压力场预测验证。
    \item \textbf{交付成果：} 封装为标准化软件包，打通 仿真→评估→预警 闭环。课题结题验收通过，获\textbf{3项发明专利+2项软件著作权}。
\end{itemize}
