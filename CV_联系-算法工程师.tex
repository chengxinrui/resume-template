% --- 科研一：UNOP ---
\textbf{UNOP：无监督神经算子用于长时序预测} 
\textit{(一作，投稿IJCAI CCF-A)} 
\hfill 2025.3 -- 至今
\begin{itemize}
    \item 针对长序列预测中\textbf{误差累积与泛化性不足}问题，设计无监督深度学习框架，通过物理一致性约束替代标注数据，实现\textbf{无监督训练范式}。
    
    \item 提出低阶积分一致性损失（LIPE），将高维场景训练显存降低至\textbf{1/6}，使大规模3D数据训练成为可能。
    
    \item 设计几何感知编码器与自适应门控机制，针对\textbf{非结构化数据}（点云/Mesh）实现高效特征提取，提升模型在不规则拓扑结构上的泛化能力。。
    
    \item \textbf{实验结果：}多场景预测误差优于SOTA方法，长序列外推步数达\textbf{50步}，（基线方法约20步发散），验证了框架的鲁棒性与可扩展性。
\end{itemize}

\textit{(发明专利已受理)} \hfill 2024.9 -- 2025.3
\begin{itemize}
    \item 针对图结构数据中\textbf{局部关键区域建模不足}问题，设计结构感知的稀疏注意力机制。

    \item 基于图稀疏结构实现\textbf{O(E)}复杂度，在保证表达能力的同时提升计算效率。
    \item \textbf{结果：}关键指标误差下降\textbf{40\%}，推理时间\textbf{20ms}，满足实时推理需求。
\end{itemize}
