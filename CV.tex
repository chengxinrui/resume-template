\textbf{汽车气动性能AI快速预测系统} 
\textit{(核心开发者，发明专利已受理)} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{业务背景：}
    汽车外流场CFD仿真（ANSYS Fluent）
    单次耗时\textbf{6--12小时}，
    造型迭代阶段需反复计算
    气动阻力系数$C_d$与压力分布，
    严重制约研发效率。
    目标：用AI模型实现
    \textbf{秒级气动性能评估}，
    支撑快速造型筛选。
    
    \item \textbf{核心难点与方案：}
    车身曲面几何复杂
    （A柱转角、后视镜、
    扰流板等局部特征），
    标准深度学习模型在
    \textbf{几何突变区域}预测误差大。
    设计\textbf{几何门控注意力机制}
    (GeoGatedAttention)，
    利用网格\textbf{法向量、曲率、
    二面角}等边级几何特征
    动态调制注意力权重，
    自适应聚焦气动关键区域。
    
    \item \textbf{工程指标：}
    \begin{itemize}
        \item 气动阻力系数$C_d$
        预测误差 \textbf{2.69\%}
        （工程可接受阈值通常5\%）
        \item $C_d$预测值与CFD真���
        相关系数 $\rho$ = \textbf{0.97}
        \item 单次推理 \textbf{20ms}，
        \textbf{替代6--12小时CFD}，
        加速比$\sim$$10^5$
    \end{itemize}
    
    \item \textbf{落地价值：}
    可嵌入造型评审流程，
    设计师调整造型后秒级获得
    $C_d$反馈，将``设计—仿真''
    迭代周期从\textbf{天级缩短至
    分钟级}。
\end{itemize}

    \item \textbf{解决的工程痛点：}
    训练AI仿真替代模型
    需先跑大量CFD生成训练数据，
    ``用仿真训练替代仿真''
    的悖论导致前期成本高。
    本工作实现\textbf{无需任何
    CFD标签数据}即可训练，
    直接从物理方程
    （N-S方程/对流扩散方程）
    学习。
    
    \item \textbf{关键技术突破：}
    \begin{itemize}
        \item 基于\textbf{积分守恒}
        约束替代微分残差，
        天然满足质量/动量守恒，
        3D训练\textbf{内存降至1/6}
        \item 适配\textbf{任意复杂
        CAD几何}（非结构化网格），
        无需规则化预处理
        \item 多工况连续预测
        \textbf{50步仍稳定}
        （现有方法约20步发散）
    \end{itemize}
    
    \item \textbf{验证结果：}
    3D汽车外流场$C_d$误差
    \textbf{4.06\%}，
    物理守恒违背量
    $2.9\times10^{-3}$。
    \textbf{可推广至热管理、
    结构强度等多物理场景。}
\end{itemize}
