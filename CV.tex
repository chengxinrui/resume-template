\textbf{复杂3D工业场景的数据闭环与自动化仿真Pipeline构建} 
\textit{(AI算法与系统集成负责人)} 
\hfill 2024.9 -- 2025.6\\[0.2em]
\textcolor{CVBlue}{\textbf{十四五国家重点研发计划 (解决真实工业长尾脏数据的自动化处理与评测)}}
\begin{itemize}
    \item \textbf{痛点与挑战：} 工业现场传感器采集的 3D 几何数据（.glb/.stl）极度混乱，存在大量非流形破损、孔洞；真实流场点云与仿真坐标系严重错配，\textbf{无法直接用于 AI 模型的训练与自动化评测}。
    \item \textbf{从零构建自动化 3D 网格与评测 Pipeline（全链路闭环）：} 
        1) \textbf{自动化拓扑修复}：利用 PyVista 提取特征边定位孔洞，编写脚本调度底层算子完成流形网格封闭；
        2) \textbf{网格质量控制}：调用 TetGen 批量将表面转换为高质量四面体体网格；
        3) \textbf{多源真实数据融合}：设计基于 KD-Tree 的 IDW(反距离权重) 空间检索算法，将散乱的百万级传感点云\textbf{精准映射/对齐到统一的 VTK 节点上}，打通真实数据与仿真环境的屏障。
    \item \textbf{跨域图数据重构与模型泛化：} 编写自定义 PyG Dataset 构建图拓扑，在不改变网络架构前提下，将基于\textbf{汽车数据集}训练的 AI 模型迁移至复杂工业场景。打通了“脏数据输入 $\rightarrow$ 自动化清洗对齐 $\rightarrow$ 模型推理 $\rightarrow$ 结果评测”的完整工程闭环。
\end{itemize}

\textbf{几何感知AI模型在汽车3D拓扑与物理场中的快速预测} 
\textit{(核心开发者，发明专利已受理)} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{工业级长尾数据Benchmark：} 在真实汽车工业数据集（DrivAerNet++）上复现多款前沿神经算子。精准定位现有模型在处理\textbf{溜背/三厢、复杂底盘等“几何突变/长尾区域”}时，极易产生局部误差飙升的瓶颈。
    \item \textbf{核心算法 (GeoGatedAttention)：} 设计几何门控注意力机制。通过提取 3D 网格的边缘曲率、法向量夹角与边长作为 \texttt{edge\_geom\_feat}，让模型\textbf{自适应感知车体长尾折角与平滑面}，提升生成/预测的真实度。
    \item \textbf{仿真加速与业务价值：} 汽车气动指标预测误差降至 \textbf{2.69\%}。单次推理耗时 \textbf{20ms}，替代传统 CFD 数小时的计算，将“设计-仿真-评测”的迭代周期从\textbf{天级缩短至分钟级}。
\end{itemize}
