\textbf{物理约束神经算子的长时序稳定预测模型} 
\textit{\textbf{一作，CCF-A}}
\hfill 2025.9 -- 至今

\textbf{背景：} 现有无监督 Neural Operator 难以同时满足无监督训练、泛化非规则几何与长时间稳定 rollout。在 3D 场景下易出现误差快速累积、梯度不稳定及显存占用过高问题。\\
\textbf{个人职责：} 主导整体模型架构设计，提出积分一致性物理约束（LIPE），完成理论推导、算法实现及 3D 非结构化几何验证。\\
\textbf{核心难点：}
    \begin{itemize}[leftmargin=1.6em]
        \item 微分残差依赖高阶空间导数计算，在非结构网格与长时间 rollout 中数值不稳定且显存开销大  
        \item FNO 等频域算子依赖规则网格，难以直接处理真实 CAD / 点云等非规则几何  
        \item 自回归预测误差随时间指数级累积，20 步后出现明显发散
    \end{itemize}
\textbf{核心突破：} 将物理约束从基于微分残差的局部约束，提升为基于积分演化一致性的算子级约束。该方法不再计算高阶空间导数，从机制上减缓误差在时间维度的指数级累积，实现无监督条件下的长时间稳定预测。\\
\textbf{效果：} 在 50-step 长时间 rollout 下预测保持稳定；1D对流扩散方程预测中相对 $L_2$ 误差由 \textbf{3.2\%} 降至 \textbf{0.8\%}；3D 非结构化几何任务中显存占用降低至 PINN 类方法的 \textbf{1/6}；10\% 标签半监督下阻力系数预测误差降至 \textbf{4.06\%}。

\textbf{基于Transformer的几何感知注意力的3D物理场预测模型} 
\textit{\textbf{(核心开发者，发明专利已受理)}} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{问题：} 针对复杂几何结构（曲率突变、边界不连续）下模型难以捕捉局部特征的问题，影响预测精度。
    \item \textbf{方法：} 设计\textbf{几何门控注意力机制（GeoGatedAttention）}，融合曲率、法向量与边结构信息，对Attention权重进行动态调制。
    \item \textbf{成果：} 1. 在3D物理场预测任务中将误差降至\textbf{2.69\%}，预测与真实值相关系数达\textbf{0.97}； 2. 构建端到端几何特征提取与训练框架，支持20GB规模（889辆汽车）非结构化网格的自动化批处理与训练，完整训练周期约10小时，推理时间仅\textbf{20ms}。
\end{itemize}
