\textbf{UNOP：面向长时序预测的无监督建模方法} 
\textit{(一作，IJCAI CCF-A 审稿中)} 
\hfill 2025.3 -- 至今
\begin{itemize}
    \item 设计无监督序列建模框架，在无标签数据条件下实现高维时序数据的稳定预测与生成，解决长时序建模中的误差累积问题。
    \item 提出新型约束目标函数替代高成本梯度计算，将3D训练显存降低至\textbf{1/6}，显著提升训练效率与可扩展性。
    \item 设计几何解耦编码器（GALA）+ 曲率感知门控（GSEO），增强模型对\textbf{非结构化3D几何}的表示与生成能力，支持任意网格拓扑。
    \item \textbf{实验结果：}在多任务场景下预测误差显著降低（最高提升36$\times$），并实现\textbf{长时序稳定外推（50步）}。
\end{itemize}

\textbf{GeoGatedAttention：面向复杂结构数据的高效图模型} 
\textit{(发明专利已受理)} \hfill 2024.9 -- 2025.3
\begin{itemize}
    \item 针对3D非结构化数据中\textbf{关键区域建模不足}问题，设计结构感知注意力机制提升模型表达能力。
    \item 引入结构信息作为门控信号动态调节注意力权重，实现特征相关性建模。
    \item 基于图稀疏结构实现\textbf{O(E)}复杂度，兼顾大规模3D数据的表达能力与计算效率。
    \item \textbf{结果：}压力场误差下降\textbf{40\%}，Cd误差\textbf{2.69\%}，单帧推理\textbf{20ms}。
\end{itemize}
