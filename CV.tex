\begin{itemize}
    \item \textbf{挑战：} 工业现场三维数据极度混乱，几何模型（.glb/.stl）存在非流形破损，多源点云数据坐标系错配，无法直接用于深度学习建模。
    \item \textbf{自动化3D数据Pipeline（全链路）：} 
        1) \textbf{拓扑修复}：利用 PyVista 提取特征边定位孔洞，结合 Meshlab 完成流形网格封闭；
        2) \textbf{网格处理}：调用 TetGen 完成体网格划分；
        3) \textbf{多源数据融合}：编写基于 KD-Tree 的空间插值算法，将散乱点云数据精准映射到统一网格节点。
    \item \textbf{图结构构建与空间特征提取：} 编写自定义 PyG Dataset，批量提取 SDF、法向量等几何特征，构建 \texttt{radius\_graph} 图拓扑，支持下游3D场景建模任务。实现跨场景零样本迁移验证。
    \item \textbf{交付成果：} 封装为标准化工具包，课题顺利结题验收，获 \textbf{3项发明专利 + 2项软件著作权}。
\end{itemize}
