\textbf{几何感知AI模型的汽车气动性能快速预测} 
\textit{(核心开发者，发明专利已受理)} 
\hfill 2024.9 -- 2025.3
\begin{itemize}
    \item \textbf{工业级Benchmark与痛点定位：} 在真实汽车工业数据集（DrivAerNet++）和 ShapeNet-Car 上复现并测试了多款前沿神经算子（Transolver等）。发现现有模型在处理溜背/三厢、光滑/细节底盘等\textbf{几何突变区域}时，难以捕捉局部边界层效应，导致预测误差飙升。
    \item \textbf{核心算法创新 (GeoGatedAttention)：} 为解决上述痛点，设计了\textbf{几何门控注意力机制}。通过底层脚本提取网格的边缘曲率($tanh$缩放)、法向量夹角($cos\theta$)与边长，作为图结构中的 \texttt{edge\_geom\_feat} 动态调制 Attention 权重。让模型自适应感知“折角”与“平滑面”。
    \item \textbf{工程指标与业务价值：} 气动阻力系数 $C_d$ 预测误差降至 \textbf{2.69\%}，相关系数 $\rho = 0.97$。单次推理 \textbf{20ms}，替代传统CFD 6-12小时的计算，将“设计-仿真”迭代周期从\textbf{天级缩短至分钟级}。
\end{itemize}

\textbf{UNOP：面向长时序时空预测的无监督神经算子} 
\textit{(一作，CCF-A 审稿中)} 
\hfill 2025.9 -- 至今
\begin{itemize}
    \item \textbf{问题与挑战：} 现有自回归时序模型在复杂3D几何下\textbf{20步即发散}；传统物理约束(PINN)在3D场景下内存爆炸且梯度病态，难以支撑长时序工程级仿真。
    \item \textbf{核心方法 (LIPE+GALA+GSEO)：} 提出基于\textbf{积分一致性}的物理约束(LIPE)替代微分残差，\textbf{无需CFD标签}即可训练，3D内存降至\textbf{1/6}。设计几何解耦编码器(GALA)统一处理不规则网格，并利用曲率感知门控(GSEO)保留激波/涡旋等高梯度时空结构。
    \item \textbf{工程指标：} 多工况连续外推 \textbf{50步仍保持稳定}。3D汽车外流场预测误差 \textbf{4.06\%}，物理守恒违背量极低 ($2.9\times10^{-3}$)。
    \item \textbf{跨域扩展性：} 其底层的长序列自回归与时空一致性建模能力，可无缝迁移至自动驾驶场景中的\textbf{未来状态预测}。
\end{itemize}

\textit{(独立在研 / 架构设计与核心验证阶段)} 
\hfill 2026.1 -- 至今
\begin{itemize}
    \item \textbf{统一连续算子架构 (Continuous Operator)：} 针对传统神经算子依赖规则网格、且 1D/2D/3D 维度割裂的痛点，引入\textbf{隐式神经卷积 (INC)} 参数化空间连续积分核，彻底替代离散卷积，实现\textbf{分辨率无关}的通用特征提取。
    \item \textbf{双域物理感知门控 (Dual-Domain Gating)：} 首创时空与频率的正交注意力。在\textbf{空间域}结合 SDF 与高斯曲率，自动聚焦高梯度边界层；在\textbf{谱域 (频域)} 构造自适应谱核，根据物理条件动态过滤/保留高频模态。
    \item \textbf{几何约束与工程落地：} 提出\textbf{统一积分守恒自监督损失}，规避复杂 3D 几何下 PINN 的梯度病态问题。具备极强的\textbf{Zero-shot 超分辨泛化能力}（粗网格训练、细网格推理），天然适配车企底层 CAE 结构应力与多物理场仿真。
\end{itemize}

    \item \textbf{交付成果：} 封装为标准化软件包，打通 仿真→评估→预警 闭环。课题结题验收通过，获\textbf{3项发明专利+2项软件著作权}。
\end{itemize}
