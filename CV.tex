\textbf{核心研究成果（第一作者）}
\begin{itemize}
    \item \textbf {Physics-Constrained Unsupervised Neural Operator for Long-Horizon PDE Learning on Generalized Geometries (\textbf{一作}，IJCAI CCF-A)}
    \begin{itemize}[label=\tiny$\bullet$, leftmargin=1em]
        \item \textbf{研究痛点：} 现有无监督神经算子依赖微分残差进行物理约束，高阶空间导数在非结构化几何与长时自回归演化中易引发数值不稳定与误差累积。
        \item \textbf{核心创新：} 提出积分一致性物理嵌入（LIPE），将 PDE 约束由微分残差重构为\textbf{积分演化形式}，在无标签条件下显著提升训练稳定性并降低显存开销。在此基础上，通过几何无关潜空间适配（GALA）统一非结构化几何表示，并构建门控谱演化算子（GSEO）以保持高频结构，实现复杂几何场景下的长时稳定预测。
        \item \textbf{实验指标：} 1D 高对流场景中 20-step 误差降至 \textbf{0.09\%}，并在 $T=50$ 外推中保持稳定；2D Navier–Stokes 任务中 20-step 误差为 \textbf{3.02\%}；3D 半监督几何任务中阻力系数误差为 \textbf{4.06\%}，显著优于基线方法。
    \end{itemize}
    \item \textbf {Integral-Consistent Basis-Universal Neural Operators for Diverse PDE Classes (\textbf{一作}，NeurIPS CCF-A, 在投)}
    \begin{itemize}[label=\tiny$\bullet$, leftmargin=1em]
        \item \textbf{研究痛点：} 传统谱神经算子将交互权重绑定于固定 Fourier 频率结构，导致模型在不同 PDE 类别（椭圆/抛物/双曲）之间难以共享动力学表示，并在长时自回归预测中产生误差漂移。
        \item \textbf{核心创新：} 提出\textbf{基表示–交互解耦的 Basis-Universal 演化框架}，将谱基表示与交互权重矩阵分离，构建与频率索引无关的统一复数潜空间交互机制，实现跨 PDE 类别的共享动力学建模。在此基础上引入几何感知潜空间调制与积分一致性约束，增强复杂几何条件下的泛化能力与长时稳定性。
        \item \textbf{实验验证：} 在跨 PDE 类验证中，1D 对流扩散任务中 20-step 误差保持在 \textbf{0.09\%} 量级；在 2D Navier–Stokes 强迫场景下 20-step 误差为 \textbf{1.90\%}；在 3D 半监督几何任务中 $C_d$ 误差降至 \textbf{3.95\%}，显著优于谱基与积分基线方法。
    \end{itemize}
\end{itemize}

\section{\makebox[\widthof{\faBriefcase}][c]{\color{CVBlue}%
\faBriefcase}\ 国家重点研发项目}
\textbf{复杂工业三维几何下的物理场建模与数字孪生研究}
\textit{\textbf{核心成员，AI算法负责人}}
\hfill 2024.9 -- 2025.6\\[0.2em]
\textcolor{CVBlue}{\textbf{“十四五”国家重点研发计划——装备制造后市场服务协同平台研发与应用}}
\begin{itemize}
    \item \textbf{研究挑战：} 理论神经算子模型在真实工业非结构化三维几何条件下易出现表示失配与稳定性退化，难以直接迁移至复杂装备场景。
    \item \textbf{算法适配与方法设计：} 针对真实工业非结构化三维几何存在拓扑缺陷与尺度异构的问题，构建统一的几何拓扑修复与潜空间表示框架，并对神经算子结构进行适配改造，在工业数据上验证其在复杂几何条件下的稳定演化与泛化能力。
    \item \textbf{真实场景验证：} 在 70GB 级工业三维数据上完成模型泛化测试，3D 物理场预测误差降至 \textbf{2.69\%}，阻力系数误差为 \textbf{4.06\%}，实现复杂几何条件下的稳定推理（20ms）。
    \item \textbf{个人职责：} 负责算法架构设计、数据建模策略制定及核心模块实现。
\end{itemize}
