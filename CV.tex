\textbf{物理约束神经算子的长时序稳定预测模型} 
\textcolor{CVBlue}{\textbf{一作，IJCAI CCF-A}}
\hfill 2025.9 -- 2026.4

\textbf{背景：} 现有无监督 Neural Operator 难以同时满足无监督训练、非规则几何泛化与长时间稳定 rollout。在 3D 场景下易出现误差快速累积、梯度不稳定及显存占用过高问题。\\
\textbf{个人职责：} 主导模型架构设计，提出积分一致性物理约束，完成理论推导、算法实现及 3D 非结构化几何验证。\\
\textbf{相关技术：} Python, PyTorch, PyG (PyTorch Geometric), Linux。\\
\textbf{核心难点：}
    \begin{itemize}[leftmargin=1.6em]
        \item 微分残差依赖高阶空间导数计算，在非结构网格与长时间 rollout 中数值不稳定且显存开销大  
        \item FNO 等频域算子依赖规则网格，难以直接处理真实 CAD / 点云等非规则几何  
        \item 自回归预测误差随时间指数级累积，20 步后出现明显发散
    \end{itemize}
\textbf{核心突破：} 提出积分一致性物理约束（LIPE），将物理约束由微分残差形式改为积分演化一致性形式，避免显式计算高阶空间导数，显著提升训练稳定性与长时间预测稳定性。\\
\textbf{效果：} 在 \textbf{50-step} 长时间预测下保持稳定，误差曲线无指数爆炸；1D 对流扩散方程中相对 $L_2$ 误差由 \textbf{3.2\%} 降至 \textbf{0.8\%}；3D 非结构化几何任务中显存占用降低至 PINN 类方法的 \textbf{1/6}；10\% 标签半监督下阻力系数误差降至 \textbf{4.06\%}。

\textbf{支持跨 PDE 泛化与长时稳定演化的神经算子框架} 
\textcolor{CVBlue}{\textbf{一作，NeurIPS (CCF-A)，在投}}
\hfill 2026.03 -- 至今

\textbf{背景：} 在解决物理约束稳定性问题后，进一步关注谱表示对跨 PDE 泛化能力的影响。当物理机制改变（如扩散转为湍流）或谱分布发生漂移时，现有模型泛化能力下降，长时演化易失稳。\\
\textbf{个人职责：} 独立负责模型表示解耦设计与动力学交互核参数化实现，主导跨 PDE 类通用性验证与分析。 \\
\textbf{相关技术：} PyTorch, Complex-valued NNs, FFT, NumPy/SciPy。\\
\textbf{核心难点：}
    \begin{itemize}[leftmargin=1.6em]
        \item \textbf{频率索引绑定：} 权重依赖固定频率位置，导致模型难以在不同 PDE 谱结构间共享动力学模式
        \item \textbf{表示与动力学耦合：} 谱表示与演化机制难以解耦，限制算子在不同几何与物理场景下的泛化能力
        \item \textbf{频谱能量漂移：} 固定权重参数化易放大或抑制特定频率，导致长时自回归预测失稳
    \end{itemize}
\textbf{核心突破：} 设计 BUNO 架构，解耦谱基表示与动力学交互核，在统一复数潜空间中共享动力学参数，消除权重对频率索引的依赖，显著提升跨 PDE 迁移能力与长时预测稳定性。\\
\textbf{效果：} 在强迫力 2D Navier–Stokes 任务中，相对误差由 \textbf{27.6\%} 降低至 \textbf{1.90\%}（实现 \textbf{14倍} 精度提升）；在跨 PDE 类验证中展现出在长时演化中频谱能量的分布稳定性。

\textbf{工业3D数据处理与数字孪生建模系统开发} 
\textit{\textbf{AI算法与系统集成负责人}}
\hfill 2024.9 -- 2025.6\\[0.2em]
\textcolor{CVBlue}{\textbf{十四五国家重点研发计划``装备制造后市场服务协同平台研发与应用''}}\\
\textbf{项目背景：} 面向工业复杂装备的数字孪生建模需求，原始3D模型存在破损、孔洞、坐标不统一及多源格式不一致等问题，难以直接用于AI训练与物理场预测。\\
\textbf{个人职责：} 负责数据处理流程设计、几何感知模型构建与系统集成部署，打通“数据清洗 → 模型训练 → 推理部署 → 可视化展示”完整闭环。\\
\textbf{相关技术：} PyTorch, Transformer, 非结构化网格处理\\
\textbf{核心技术与实现：}
\begin{itemize}[leftmargin=1.6em]
    \item \textbf{3D数据自动化处理：} 构建非结构化网格清洗与重建流程，实现拓扑修复、坐标对齐及多源数据融合，使工业级3D模型可直接用于AI训练。
    \item \textbf{几何感知注意力建模：} 设计几何门控注意力机制（GeoGatedAttention），融合曲率、法向量与边结构信息，对注意力权重进行动态调制，增强复杂几何区域的局部表达能力。
    \item \textbf{规模化训练与部署：} 构建端到端几何特征提取与训练框架，支持约20GB规模（889辆汽车）非结构化网格的自动化批处理训练；完整训练周期约10小时，单样本推理时间约20ms。
    \item \textbf{系统开发与集成：} 基于Python开发模型管理与可视化系统，构建包含2000+工业部件的3D模型库，支持模型查询、预测调用与结果展示。
\end{itemize}
\textbf{项目效果：} 在3D物理场预测任务中将相对误差降低至 \textbf{2.69\%}，预测结果与真实值相关系数达 \textbf{0.97}，实现复杂工业结构场景下的稳定建模与快速推理。
